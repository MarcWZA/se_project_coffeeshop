# Triple Peaks Coffee Shop

This is the second project in the TripleTen Software Engineering program. The project is a single-page website for the Triple Peaks Coffee Shop, a cafe located inside the Triple Peaks Library. It was built from a design brief using HTML and CSS.

## Project description

The website introduces the coffee shop, shares coffee recipes, allows visitors to book a table, displays the cafe menu, and provides contact and social media information. The layout follows the project brief closely, including the required spacing, typography, colors, backgrounds, and section structure.

## Project features

- Semantic HTML5 structure
- CSS styling with a flat BEM methodology
- Flexbox layout
- Positioned decorative and content elements
- Embedded recipe videos
- Custom reservation form with required fields
- Menu cards with item names, prices, and divider lines
- About section with layered content and a CSS animated circle
- Footer with logo, social media links, and copyright
- Hover states and smooth CSS transitions
- Local image and SVG assets

## Technologies used

- HTML5
- CSS3
- Flexbox
- BEM naming convention
- CSS transitions
- CSS animation with `@keyframes`
- Normalize.css
- Google Fonts

## Plans for improvement

- Add responsive layouts for tablet and mobile screen sizes.
- Improve form behavior with validation messages and a submission confirmation.
- Add more menu categories and seasonal drinks.
- Include accessibility improvements such as clearer focus states and additional ARIA labels where useful.
- Connect the reservation form to a backend service.
